# Citas Medicas - Backend

API construida con [NestJS](https://nestjs.com/).

## Requisitos

- Node.js 20+
- pnpm

## Configuración

Copia el archivo de variables de entorno de ejemplo y ajusta los valores según tu entorno:

```bash
cp .env.example .env
```

## Instalación

```bash
pnpm install
```

## Ejecutar

```bash
pnpm start:dev
```
